
# Job Street web scraper using scrapy library.




## Run Locally

Clone the project

```bash
  https://github.com/cenizastevie/job_street_scrapy
```

Go to the project directory

```bash
  cd job_street_scrapy
```

Build docker image

```bash
  docker build -t job_street_scrapy . 
```

Start the server

```bash
  docker run -d job_street_scrapy
```
